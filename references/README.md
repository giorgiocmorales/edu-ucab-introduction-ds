# References Directory

This directory defines the structure for reference material used to design and adapt lectures.

Only this `README.md` is tracked in Git. The actual reference files are intentionally local-only.

Typical local subfolders include:

- `books/`: primary books and PDFs used to align lecture content
- `irizarry/`: notes derived from Irizarry that support the redesigned course
- `analytics-edge/`: notes and extracted ideas from MIT Analytics Edge
- `legacy-course/`: imported source material from the inherited course
- `style-references/`: slide decks or screenshots that capture the preferred lecture style

Why these files are excluded:

- some sources are copyrighted
- some files are bulky and not useful for normal Git history
- some notes are working material rather than canonical course output

On a new machine, recreate the folder structure and copy the local files from your private archive after cloning the repo.

Reference materials here inform new lectures, but `lectures/` remains the canonical source of course content.
