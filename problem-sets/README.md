# Problem Sets

Problem sets are primary course artifacts parallel to lectures.

Use `problem-sets/_templates/` for starter files. Individual assignments should
use numbered folders such as:

```text
problem-sets/01-r-basics/
  problem-set.qmd
  assets/
```

Use `.R` templates only when the assignment is intended to be completed as a
script. Use `.qmd` templates when students should render a document.

Generated files belong under `outputs/problem-sets/`, not inside source
folders.
