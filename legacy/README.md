# Legacy Directory

This directory stores inherited or superseded course material.

- `original-scripts/`: legacy R scripts from the previous course structure
- `original-reports/`: legacy R Markdown reports and generated report artifacts
- `original-notebooks/`: archived notebooks from the previous structure
- `original-slides/`: reserved for imported original slides if they need to be archived directly

Files in this directory should be treated as reference material, not as the canonical source for the redesigned course.
